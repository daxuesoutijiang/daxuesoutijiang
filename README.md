## 😮 主要功能

- [x] 视频、音频、文档快速播放，考前复习快人一步
- [x] 任务点自动切换，无需手动切换章节，学习更专注
- [x] 便捷搜索试题答案，整合海量题库、AI问答等平台的试题答案及解析，搜题答疑更高效
